**Learning Web** là một nền tảng học tập trực tuyến toàn diện, cung cấp công cụ quản lý khóa học, theo dõi tiến độ học viên và tích hợp lưu trữ đám mây. Dự án được thiết kế để hỗ trợ cả giáo viên và học viên trong quá trình học tập.

## Tính Năng Chính

- 📚 **Quản lý khóa học** - Create, edit, organize courses by category
- 📊 **Tracking tiến độ** - Real-time progress monitoring for students
- 📁 **Cloud storage** - File upload với integration to cloud services
- 👨‍🏫 **Multi-role system** - Teacher, student, admin roles
- 📈 **Analytics dashboard** - Detailed insights on learning patterns
- 🎓 **Certificate generation** - Automated certificate on course completion

## Tech Stack

- **Backend**: Java Spring Boot, Node.js
- **Database**: PostgreSQL
- **Frontend**: Modern UI/UX
- **DevOps**: Docker, Kubernetes
- **Build Tool**: Maven
- **Cloud Storage**: Cloud integration for file management

## Kiến Trúc Hệ Thống

```
Frontend (Next.js/React)
    ↓
Node.js API Gateway
    ↓
Java Spring Boot Services
    ├─ Course Service
    ├─ User Service
    ├─ Progress Tracking
    └─ Analytics Service
    ↓
PostgreSQL Database
```
