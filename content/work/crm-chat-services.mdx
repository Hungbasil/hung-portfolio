**CRM Chat Services** là một hệ thống quản lý quan hệ khách hàng (CRM) tích hợp tính năng nhắn tin thời gian thực. Dự án được xây dựng để giải quyết nhu cầu giao tiếp hiệu quả giữa các nhân viên support và khách hàng.

## Tính Năng Chính

- 💬 **Nhắn tin thời gian thực** - WebSocket integration cho tin nhắn instant
- 👥 **Quản lý người dùng** - Multi-user support với role-based access
- 📊 **Lưu trữ hội thoại** - Persistent database cho chat history
- 🎯 **Schema linh hoạt** - Cấu trúc dữ liệu dễ mở rộng
- ⚡ **Server ổn định** - High-performance API endpoints

## Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Real-time**: WebSocket
- **Deployment**: Docker

## Kết Quả

Hệ thống đạt được:
- Phản hồi tin nhắn < 100ms
- Support tới 500+ concurrent users
- 99.5% uptime
- Easy maintenance & scalability
